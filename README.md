# 📖 Overview

**StreamPulse** is a modern IPTV streaming dashboard with a cyber-inspired interface. It allows users to browse live TV channels, search by category, and watch HLS/DASH streams through a clean, responsive interface powered by **Video.js**.

Designed with performance and usability in mind, StreamPulse combines a premium UI with lightweight JavaScript to deliver a smooth streaming experience across desktop and mobile devices.

# 🛠 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/StreamPulse.git
```

Go to the project

```bash
cd StreamPulse
```

Install dependencies

```bash
npm install
```

Run Netlify Development

```bash
netlify dev
```

Open

```text
http://localhost:8888
```
